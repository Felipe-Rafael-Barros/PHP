Como rodar?

1) Abra o Devserver17
2) http://127.0.0.1:1111/ ou abre no dashboard
3) Iniciando os servidores( HTTP Server(Apache)  e Database Server(MySql/MariaDB))
4) Padrao: C:\Program Files (x86)\EasyPHP-Devserver-17\eds-www\
5) C:\Program Files (x86)\EasyPHP-Devserver-17\eds-www\meuprojeto\
6) Seu codigo: http://localhost/meuprojeto/


O PHP é uma tecnologia de complemento para o HTML e CSS
